# Tetris Game

This is a browser-based implementation of the classic Tetris game. The game is built using HTML, CSS, and JavaScript, and it features a fully interactive grid, score tracking, and a preview grid of the next tetromino.

## Features

- **Interactive Gameplay**: Move, rotate, and drop tetrominoes using keyboard controls.
- **Score Tracking**: Earn points by clearing rows and placing the tetrominoes strategically.
- **Next Tetromino Preview**: See the next tetromino in the mini-grid.
- **Pause and Resume**: Pause the game and resume at any time.
- **Game Over Detection**: The game ends when the tetrominoes can no longer be placed.
- **Responsive Timer**: Tracks the elapsed time during gameplay.
- **Shadow Tetromino**: Displays a shadow of where the tetromino will land.

## How to Play

1. **Start the Game**: Click the "Start/Pause" button to begin.
2. **Controls**:
   - **Move Left/Right**: ← / → Arrow keys.
   - **Move Down**: ↓ Arrow key (soft drop).
   - **Rotate**: ↑ Arrow key.
   - **Hard Drop**: Spacebar.
   - **Pause/Resume**: Escape key or "Start/Pause" button.
3. **Place Tetromino**: Placing tetrominoes grants points.
4. **Clear Rows**: Clearing full rows removes them and increases your score.
5. **Game Over**: The game ends when tetrominoes reach the top with no valid placements left.
